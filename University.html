General Information


Undergraduate studies on behalf of the study program of FINKI aim to provide top engineering staff to meet the growing demand for computer engineers. Students in this study program gain knowledge that provides them with great professional flexibility and a wide selection of jobs wherever there is a need to design, implement and maintain computer systems. The graduate engineer of this profile can build his career in industry, in the public and service sector, but, especially in the companies for development of computer systems, systems that need to work in real time, systems for support of telecommunication services, services based on Internet, advanced user interfaces, application solutions related to the design and implementation of computer systems and more.

Structured Programming
To introduce students to the paradigm of structural programming, to understand the concept of algorithms and to be able to develop algorithms, code, test and compile programs. Data types, control structures, functions, fields, files will be introduced.
Course content: Introduction to the subject, programming languages. Method of writing numbers in computer, concept of structural programming, variables, first program. Operations / arithmetic / relational, logical. Control structures, if-else, cycles,. algorithms (correct algorithmic thinking and program development). Arrays, Functions, Character Arrays, Functions 2 / Recursion, Files, Integrated Example Processing.
Coding Language: C 6 ECTS

Object Oriented Programming
The aim of the course is to acquaint the student with the basic concepts of object-oriented programming. For this purpose, the concepts of objects and classes, encapsulation, inheritance and polymorphism will be introduced. Students will also be introduced to the hierarchy of classes. Comparisons of object-oriented concepts implementations in different programming languages will be made. Upon completion of the course the student will understand the principles of object-oriented programming and be able to develop programs that support these principles.
Course content: Introduction to object oriented programming / CSR concepts, Objects, Classes, Constructors and Destructors, Classes 2, copy constructor, this pointer, overload operators, Inheritance, Polymorphism, Static members of classes, Multiple inheritance, Exceptions, Practical example processing. 6 ECTS Coding Language: C++ 6 ECTS

Algorithms and data structures
Introduction to the basic data structures and algorithms that are further necessary for work with databases as well as for other applications. The student will be able to use and development of structures and algorithms with linear lists, trunks, graphs as well as indexes for search. You will also be trained in the implementation of various archetypes of algorithms used in the practical implementation of many software solutions.
Course content: Introduction to data structures. Algorithm analysis and complexity of algorithms. Representation of data with fundamental data structures (arrays and lists). Introduction to algorithms and techniques for creating algorithms (brute force, greedy, divide-and-conquer, dynamic programing…). One-dimensional data structures (warehouse, queue). Algorithms for sorting. HASH structures. Hierarchical structures - trees. Graphs
Coding Language: Java 6 ECTS

Operating Systems
The student will be introduced to the basic building blocks of modern operating systems through their concrete implementation in Windows and UNIX-like systems.
Course content: Overview of operating systems, role, history, functionality; Definitions of operating systems; Processes and process and thread management; Competitiveness, states and state diagrams, implementation structures; Resource allocation, principles of allocation, management and dismissal of events; deadlock; Memory management, types of memory devices, partitioning. Device management, devices with serial and parallel connections, direct memory access, buffering strategies. File system, basic concepts, content and structure of directories, memory mapped files, naming, search and access, protection strategies. Basic elements of security and protection, overview, methods and devices, access and authentication, protection models, memory protection, description of a specific OS. Input and output devices and communication
Coding Language: Java, UNIX (and shell-scripts)

Software Engineering
The aim of the course is to get acquainted with: methodologies for design and implementation of software systems through requirements analysis, development and analysis of design, implementation, integration and testing of software; problem analysis for software development, designing a modular solution structure that solves the problem and analysis and evaluation of alternatives; implementation of efficient and correct modules, their integration and testing, planning and management of software projects.
Course content: Introduction to software engineering. Software features and software projects. Software development process and methodologies: decomposition, abstraction, object model. Analysis and modeling of requirements: structured analysis, object-oriented analysis, formal specifications. Software design and software architectures: writing specification, conceptual, logical and physical design of software. Templates for design, static and dynamic analysis, implementation, documentation, planning. Introduction to testing: functional testing, systematic testing. Running and managing software projects
Coding Language: Software testing - Gradle, Java 6 ECTS

Web Design
Upon completion of the course, students are expected to receive introductory knowledge for developing web pages with contemporary design, use of HTML and cascading styles. Candidates are expected to: 1. Demonstrate a basic understanding of the importance of good web design. Demonstrate practical knowledge of web design technologies and be able to apply knowledge in basic web site design. 3. Communicate with terminology specific to this area. 4. Compatibility with earlier versions of HTML and XHTML. Cascading styles - CSS3, properties, length units, selectors, pseudo-elements, model box, Positioning, margins, page layout organization - methods Introduction to site design, methodologies, structure and architecture, navigation, Functionality, Web Writing, Typography, Color, Design Concepts, Gold cross section, contrast. Basics of Usability, Basics of Accessibility
Languages : HTML , CSS 6 ECTS

Design of user interface
The aim of the course is to introduce students to the basic problems and ways to solve them for different types of user interfaces and user interaction. Upon completion of the course, the student is expected to demonstrate knowledge of the basic types of user interaction and the principles for their design, and to be able to independently develop interactive applications using programming tools following the learned principles.
Course content: Introduction to the subject. Customer behavior. Introduction, brief history, user behavior when working with interactive systems, templates for describing user behavior. Content organization in interactive systems. Organizational schemes, structure organization, templates to organize and display data. Navigation. Problems. Principles for good navigation. Navigation models. Navigation design templates in interactive systems (navigation models, combining layout and navigation models, bookmarking, animated transitions). Organization on the screen. Basic concepts (visual hierarchy, visual flow). Techniques for grouping and ordering elements. Gestalt principles. Templates for organizing screens. Forms and data entry. Design principles of forms. Types of data entry elements and controls. Templates for designing shapes (relations between elements, values of elements…) Actions and commands. Stock rendering. Representation of shares. Stock templates and commands (display of shares, improve stock selection, information about long-standing actions, working with sequences of actions). Interact with data lists. Information architecture. Interaction models. Interaction in social media. Principles and manner of communication with and on social media. Interaction models. Interaction in mobile devices. Features that determine the interaction in mobile devices. Interaction models. Work with API, AJAX
Languages : HTML , CSS, Javascript, Jquery 6 ECTS

Interactive Web Aplications
The aim of the course is to introduce students to the basics of programming on the Internet client side. In this regard, students will be introduced to some of the programming languages and programming technologies that run on the client side. Upon completion of the course, the student is expected to be able to develop interactive web pages using programming languages designed for client-side programming.
Course content: Introduction to the subject, basic concepts for client-side programming, rules for writing programs for the client side. Web architecture applications. Introduction to scripting. JavaScript - variables, control structures, functions, fields and objects. Document Object Model. JavaScript - handling errors and exceptions, events. JQuery - syntax, selectors, methods, application for adding interactivity. JSON - syntax and data exchange application. Ajax - server side connection and data exchange.
Languages : HTML , CSS, Javascript, Jquery 6 ECTS

Digital logic circuits
Introduction to the basic methods for analysis and design of logic circuits and logic networks, both combinational and sequential circuits, registers and counters.
Course content: Numerical systems, base system r, conversion. Complement, first, second, radix. Boolean arithmetic. Codes, Weight, Gray, BCD, ASCII, Hamingovo distance, Hamming code. Boolean algebra and switching functions, logical gates, minterms, maxterms, levels of integration. Minimization of functions with Carnow maps and Quinn McLaughlin. Analysis and design of combination circuits, truth table, semi-adder, full adder. Encoders, decoders, multiplexers and demultiplexers, ROM, PLA. Synchronous and asynchronous sequential circuits, clock generators, latches and flip-flops, characteristic tables and equations. Synchronous analysis and design sequential circuits, equations, table and state diagram, Moors and Millie automatics, state number minimization, state coding. Registers, parallel and serial charging, moving and counters, module counters, circular meters, Johnson counter
Simulations made using: Logisim simulator 6 ECTS

Calculus 1
The subject is supportive and is necessary for the introduction of the notions of function, limit functions which are necessary in almost all subjects of higher years.
Course content: Definition of a function. Properties of functions. Function operations. He does. Function families. Inverse functions. Inverse trigonometric functions. Exponential and logarithmic functions and their inverse functions. Limes. Calculation of limits. Continuity. Definition of statement. Techniques of differentiation. Extract of Complex Function. Approximation with tangent and differential. Lopital rule. Application of derivatives: monotonicity of functions, convexity and concavity of functions, local extremes. Examination of properties and sketching a graph of a function. Global extremes.
Simulations made using: Geogebra 6 ECTS

Calculus 2
The course is supportive and is necessary for the introduction of the terms integrals, application of integrals, sequences and rows that are necessary in some of the courses of higher years.
Course content: Integration: indefinite integral, substitution integration. Certain integral. Fundamental theorem in calculus. Integration techniques: partial integration, trigonometric integrals, trigonometric shifts, integration of fractional-rational functions. Application of integrals for calculation of: area, volume, length of the arc of a curve, area of rotational surface. Arrays and rows. Differential equations: solving equations with separable variables, first-order linear differential equations, homogeneous and inhomogeneous second-order linear differential equations.

Computer Architechture
Understanding the architecture of a computer system, identifying its components, evaluating the performance of the computer system and its individual components. Understand machine language, assembler and memory organization of systems.
Course content: Computer architecture and organization. Structure and function. Evolution and performance. Von Neumann machine. Moore's Law. Problems with tact and density on logic gates. Embedded systems. Benchmarks, SPEC, Amdalov Law. Representation of integers. SM, RC, DC systems. Arithmetic operations in systems. Floating point numbers. Global view and interconnection. Program concept. Hardware and software programming. Instructional cycle. Interrupt. Types. Computer modules. Connection, Bus. Types of buses and arbitration, type of transfer, synchronous and asynchronous operation . Instruction set, features and functions. Elements of instruction. Instruction formats. Design decisions. Types of x86 operands vs ARM. Using a warehouse. Order of bytes. Addressing modes and formats. X86 vs ARM Assembler. Processor structure and function. Internal structure. Organization of registers. Flow rate. Hazardi. Dealing with a grenade. RISC vs CISC. Work with a large number of registers, registration windows. Instruction-level parallelism. Superscalar processors. Superpipline. Dependencies and effects. Control unit. Micro operations. Sequence of execution of instruction cycle. Functions of control unit. Control signals. Internal organization. Internal memory. Memory features. Methods of access. Physical types. Hierarchy. Semiconductor memory. Memory organization. Detection and error correction. Cache memory. Reference locality. Cash organization. Types of cache. Mapping and addressing. Algorithms for replacement. Enrollment policies. X86 vs ARM. Virtual memory. Multiprogramming. Memory management. Swapping, partitioning, paging, TLB. Pentium vs ARM segmentation External memory. Types. Discs. Optical media. LTO magnetic tape, flash memory, SSDs Multi-core, multi-processor systems clusters, multi-threaded operation, superscalarity, types of parallelism, GPU, network topologies for multiprocessor systems.
Coding language: MIPS 6 ECTS
